# @ai-emart/mindr-core

Internal shared library for [Mindr](https://github.com/ai-emart/mindr). Contains the git watcher, tree-sitter convention detector, memory storage backends (SQLite + Remembr), session context builder, bug fingerprinter, debt detector, and quality scorer.

**This package is the internal engine. Most developers should install [`mindr`](https://www.npmjs.com/package/mindr) (CLI) or [`@ai-emart/mindr`](https://www.npmjs.com/package/@ai-emart/mindr) (SDK) instead.**
